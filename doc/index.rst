jsonrpcclient
*************

Make `JSON-RPC <http://www.jsonrpc.org/>`_ requests.

Protocols
=========

Select the required communications protocol:

- `HTTP <http_server.html>`_
- `ZeroMQ <zmq_server.html>`_
